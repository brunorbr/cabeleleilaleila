## Compatibilidade

- Mac
- Linux
- Windows

## Rodando com [npx](https://www.npmjs.com/package/npx)

> O npx vem junto com o npm a partir da versão 5.2.0.

Rode o comando:

    npx cabeleleilaleila

## Rodando via CLI

Instale globalment e rode o comando:

    npm install -g cabeleleilaleila

    leila
